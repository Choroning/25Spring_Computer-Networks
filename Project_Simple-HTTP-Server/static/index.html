Simple HTTP/1.1 Server

Built with Python sockets

About This Server

This is a basic HTTP/1.1 server implemented using raw Python sockets. It supports GET and POST requests, serves static files, and uses multi-threaded connection handling.

Features

  • HTTP/1.1 request parsing (method, headers, body)
  • GET and POST request handling
  • Static file serving (HTML, CSS, JS, images)
  • Proper HTTP status codes (200, 404, 405, 500)
  • Multi-threaded: one thread per connection
  • URL routing with path matching
