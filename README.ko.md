## 📝 레포지토리 소개

이 레포지토리에는 대학 수준의 컴퓨터네트워크 과목을 위해 작성된 자료가 포함되어 있습니다:

- 강의 노트 및 프로토콜 분석
- 학생 발표 자료 (DHCP, NAT, ARP, DNS, VPN, Tunneling)
- 추가적인 연습 문제

본 과목은 데이터 링크 계층부터 응용 계층까지 TCP/IP 프로토콜 스위트를 다루며, IP 주소 체계, 라우팅, 전송 프로토콜, 네트워크 보안을 포함합니다.
